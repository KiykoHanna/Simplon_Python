# Exercices et Projets – Formation Python & Data
Ce dépôt regroupe l’ensemble de mes exercices, projets pratiques et mini-applications réalisés dans le cadre de ma formation en Python, data science et outils associés.
Chaque dossier correspond à un module ou à un projet autonome avec son propre code, ses notes et parfois des tests automatisés.
---
### Sommaire Projet_Exersices_Python
1. Numpy
    - Manipulation de tableaux
    - Exercices d’initiation
    - Opérations vectorisées, statistiques, reshaping

2. Pyplot
    - Premiers graphiques avec matplotlib.pyplot
    - Exercices : courbes, histogrammes, scatter plots

3. Mini-jeux Python
    - Le Pendu
        - Objectifs pédagogiques :
            - Conditions, boucles, chaînes de caractères
            - Séparation logique du code en fonctions
            - Gestion des essais, affichage du mot masqué
            - Déroulement du projet :
            - Choix aléatoire d’un mot
            - Boucle de jeu avec prise d’entrée utilisateur
            - Détection victoire / défaite

    - Pierre-Feuille-Ciseaux
            - Utilisation du module random
            - Comparaison des choix joueur/ordinateur
            - Gestion du score

    - Mini-jeu de devinettes
            - Génération d’un nombre aléatoire
            - Boucle avec feedback “trop haut / trop bas”

    - Jeu de Morpion (Tic-Tac-Toe)
        - Objectif : programmer un jeu complet en terminal.
        - Fonctionnalités :
            - Affichage de la grille
            - Tour par tour
            - Détection de victoire
            - Match nul
            - Fonctions suggérées :
            - afficher_grille()
            - jouer_coup()
            - verifier_victoire()

4. Pytest
   - Mise en œuvre :
        - Installation dans un environnement virtue
        - Structure recommandée des test
        - Exemple d’un test simple
        - Astuces : gestion du chemin, organisation des modules.

5. SAS IA – Github Actions
    - Mise en place d’un workflow CI minimal
    - Exécution automatique des tests via GitHub Actions

6. Pandas
    - Manipulation de DataFrames
    - Sélection, filtrage, opérations CRUD
    - Exercices appliqués: Pandas – Mon premier CRUD
        - Create / Read / Update / Delete sur un DataFrame
        - Cas pratiques (données de ventes, series temporelles…)

7. Seaborn
    - Visualisation statistique
    - Histogrammes, boxplots, pairplots
    - Exercices sur jeu de données sales_data
